# ZhKeyesIM

## 项目简介

本项目旨在开发一款即时通讯软件，将平时所学的各项基础知识加以综合应用。系统架构如下：

1. **整体框架**
   - **服务端**：基于 Linux 平台，采用 TCP 协议和 Reactor 模型，实现账号服务、消息服务、文件传输服务等核心功能；
   - **客户端**：基于 Windows 平台，采用三层架构——表现层、数据处理层和网络层；为后续跨平台支持，客户端仅使用 Qt 进行界面渲染，不依赖其网络或线程模块。
2. **分阶段演进**
   - **阶段一**：将所有功能集成到单一服务程序中，一次性实现账号管理、即时消息、离线缓存、文件传输等必备特性；
   - **阶段二**：拆分为若干微服务（如认证服务、聊天服务、文件服务），各模块之间通过 RPC 或其他轻量协议互相调用，以增强系统的可扩展性与可维护性。
